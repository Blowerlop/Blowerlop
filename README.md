## 🚀 Featured Projects

### 🎮 [Zodiaq](https://github.com/Blowerlop/Zodiaq) — Boss Rush Game *(Unreal Engine / C++)*
> 3rd year bachelor project at Cnam-Enjmin

Online multiplayer boss rush game for 1–4 players. Defeat the corrupted divine **Taurus Wraithraiser** by figuring out boss patterns and adapting with your team.

[![Trailer](https://img.youtube.com/vi/YPPt4yNKmbs/hqdefault.jpg)](https://www.youtube.com/watch?v=YPPt4yNKmbs)

---

### 🎮 [Convergence](https://github.com/Blowerlop/Convergence) — Cross-Engine MOBA *(Unity / Unreal Engine / C#)*
> 2nd year bachelor project at Cnam-Enjmin

Multiplayer MOBA (2–8 players) that is **cross-engine** (Unity ↔ Unreal Engine) and **cross-platform** (PC ↔ Mobile). Bridges Unity and Unreal Engine clients using **gRPC** and **Netcode for GameObjects**.

[![Gameplay](https://img.youtube.com/vi/B-Pr4Q9twxA/hqdefault.jpg)](https://www.youtube.com/watch?v=B-Pr4Q9twxA)

---

### 🎮 [Chroma Clash](https://github.com/Blowerlop/Chroma-Clash) — First-Person Multiplayer Shooter *(Unity / C#)*
> 1st year bachelor project at Cnam-Enjmin

Fast-paced first-person multiplayer game where invisible players paint each other to score points. Features invisibility cloaks, dashing, and a scoring system.

[![Trailer](https://img.youtube.com/vi/E9fUczOYMlA/hqdefault.jpg)](https://www.youtube.com/watch?v=E9fUczOYMlA)
